Look for ⚡ in Activity Bar
After restart, check the vertical icon bar on the left side of VS Code. You should see a lightning bolt icon between the existing icons.
👁 Look: Left sidebar → ⚡ icon
Expected: Lightning bolt visible, click opens "PowerPlay / Prompts"

Check Status Bar (bottom-right)
The very bottom bar of VS Code should show a cyan-highlighted item showing the PowerPlay version number.
👁 Look: Bottom-right → ⚡ PowerPlay v3.9.0

Test Command Palette
Open VS Code's command palette and type "PowerPlay" — you should see all 6 registered commands appear in the dropdown.
