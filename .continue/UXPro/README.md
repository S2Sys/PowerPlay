A comprehensive, production-ready UI/UX design system for AI-powered coding assistants. Works with Claude Code, Cursor, Windsurf, Cline, Continue.dev, and any LLM-based code generator.

## 🚀 Quick Start (30 Seconds)

1. **Copy** the system prompt from `core/UIUX-SYSTEM-PROMPT.md`
2. **Paste** into your AI tool's custom instructions (see [platform guides](./docs/guides/) for details)
3. **Start** using the design system in your prompts

👉 **[Choose Your Platform Guide](./docs/guides/README.md)** for step-by-step installation

## ✨ What's Included

**30 Comprehensive Sections** covering:
- **Core Design** - Color rules, typography, spacing, responsive breakpoints
- **Components** - Buttons, forms, cards, modals, navigation patterns
- **Accessibility** - WCAG 2.1 AA compliance, keyboard navigation, ARIA labels
- **Motion** - Animation principles, GPU-accelerated transitions, easing curves
- **Patterns** - Forms, data visualization, error states, loading states
- **Advanced Topics** - State management, real-time collaboration, system updates
- **Performance** - Layout optimization, image handling, animation efficiency
- **Platform Guides** - iOS, Material Design 3, desktop, PWA considerations

**Built-in Quality**:
- ✅ WCAG 2.1 AA compliant (4.5:1 contrast minimum)
- ✅ Mobile-first responsive (320px - 1536px+)
- ✅ 8-point spacing grid system
- ✅ CSS design tokens (colors, typography, spacing)
- ✅ Semantic versioning with migration guides

## 💡 What You Can Do

With UXPro, you can:

- Generate accessible UI components that pass WCAG audits
- Create responsive designs that work across all screen sizes (320px+)
- Build consistent designs using semantic color tokens and 8-point grid
- Design micro-interactions and animations with clear principles
- Handle complex patterns (forms, data tables, navigation)
- Implement platform-specific designs (iOS, Material, desktop)
- Debug design issues with clear accessibility and performance guidelines
- Maintain design consistency across multiple projects
